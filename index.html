2048 Replica

2048 is a game where you merge tiles with equal number which creates the sum of the two. You do this until you run out of space. The game I have created runs on any browser, using technologies such as Object-Oriented JavaScript, CSS and HTML. The game saves progress in local storage on the browser, so you can come back and continue where you left the game. Your best score will also be saved in local storage so you can improve. ✌️ github

Esportsjobs.co

EsportsJobs.co is a job board for jobs in the Esports industry. I am adding jobs from different api’s and adding them to the site. Using phantomjs to fetch new jobs every hour I add them to site, and add appropriate tags based on location and required skills. Other businesses can then advertise their own jobs, in order to reach a bigger audience. The site is built using the Ruby on Rails framework, PhantomJS, Javascript, HTML, SASS and Bootstrap. I am a big fan of the Ruby language, because of its simplicity and ease of use.
